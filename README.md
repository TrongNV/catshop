Coinbolt Cat Shop
=================

Why?
----

To test your Bitcoin testnet wallet.


Install
-------

clone repo

install deps:

    npm install

run the reload local webserver:

    npm run reload

run the watch build:

    npm run watch-build

compile production build (you'll need Google closure):

	npm run build
